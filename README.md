## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar **lógica de programação** e conceitos fundamentais da linguagem C.

O sistema simula um controle de estoque, permitindo o cadastro, armazenamento e visualização de produtos diretamente pelo terminal.

---

## 🚀 Funcionalidades

✔️ Cadastro de produtos  
✔️ Armazenamento de código, quantidade e preço  
✔️ Listagem de produtos cadastrados  
✔️ Cálculo do valor total em estoque  
✔️ Controle de limite de produtos
